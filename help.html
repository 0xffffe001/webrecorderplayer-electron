What are Web Archives (WARCs)?

A Web Archive, or WARC, is a standard for capturing and organizing all kinds of web resources, such as HTML, Javascript, images and videos.

Web Archives files can have various extensions. Webrecorder Player currently supports the following:

  • WARC format (.warc, warc.gz)
  • ARC format (.arc)
  • HAR format (.har)
